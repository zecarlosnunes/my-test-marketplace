# Generate Description Skill

This skill formats the analyzed Git differences and context into a readable, professional Pull Request description template.

## Guidelines:
1. Include sections for:
   - **Summary of Changes**
   - **Motivation / Why**
   - **Detailed Breakdown**
   - **How was this tested?**
   - **Screenshots or Logs** (if applicable)
2. Use markdown lists and checkboxes for clean layout.
